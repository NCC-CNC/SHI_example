  per-species breakdown stays near the map. ShiSummary gained an optional
  beforeOverall prop.

Edit-grid keyboard accessibility (PR #52, closed #41)

- Replaced ~900 per-cell tab stops with the roving-tabindex grid pattern: the
  grid is a single tab stop (role="grid"), arrow keys move a cursor, Home/End
  jump to row ends, Enter/Space paints. aria-activedescendant + per-cell
  role="gridcell"/aria-label for screen readers. Focus ring on the container,
  keyboard cursor on the active cell. Mouse hover/click unchanged.

## Current state

What works:

- Full pipeline live and interactive on the generated ~30x30 landscape, plus the
  new layered documentation. Three ways to learn the concepts: the seven-step
  guided tour (walkthrough), the About modal (reference), and inline "?"
  tooltips (per-term).
- Editing is now keyboard and screen-reader accessible: one tab stop, arrow to
  move, Enter/Space to paint.
- Change impact sits under the map; the top index readout shows a live delta
  versus the pre-edit state whenever there are edits.
- 90 tests pass; CI green; deploy automated.

Architecture (unchanged shape):

- src/engine/ pure SHI logic (no React), unit-tested. src/data/ typed content
  plus landscape.ts generator feeding scenario.ts. src/ui/ React components,
  share-state.ts, tour.ts, and the new AboutModal.tsx / InfoTip.tsx.

Verification this session used a real browser (Playwright + Chromium, installed
--no-save so it is not a project dependency) to confirm the About modal, an open
tooltip, the always-visible delta after edits, and keyboard-only grid editing
(painted cells with arrows + Enter, focus ring and active-cell marker visible).

Process note / gotcha:

- CI runs `format:check` (Prettier) as a separate gate from `lint`. PR #48 first
  failed on it. Run `npm run format:check` (or prettier --write) before pushing;
  this was done for every PR after #48.
- Local main can go stale between merges. After a squash-merge, run
  `git pull --ff-only` on main before branching the next slice. Mid-session a
  branch was cut from a stale main and briefly lost merged files; caught before
  committing, recovered by discarding, pulling, and rebuilding on fresh main.

Known limits / simplifications (by design, unchanged):

- Synthetic data only; no real ESA CCI, range, or elevation clipping. Quality
  folded into suitability. Simple-mean aggregation. Grid is a generated ~30x30
  on SVG (not 100x100; that would need a canvas renderer).
- Visual checks are still manual (Playwright screenshots reviewed by the agent);
  no in-repo visual regression testing.
